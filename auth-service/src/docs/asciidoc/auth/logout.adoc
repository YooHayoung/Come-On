[[Logout]]
== 로그아웃

POST /auth/logout

[[Logout-Normal]]
=== 정상 흐름

로그아웃 기능은 아래 조건을 만족하면 정상적으로 처리하고 +
해당 Access Token과 사용자의 Refresh Token을 사용할 수 없게 합니다.

----
* 인증 헤더에 Bearer 타입의 유효한 AccessToken을 담아 전송한다.
----

==== 요청 예시

include::{snippets}/logout/success/http-request.adoc[]

==== 요청 헤더

include::{snippets}/logout/success/request-headers.adoc[]

``AccessToken``이 유효하다면 로그아웃을 진행합니다. +
로그아웃을 성공적으로 수행하였다면 다음과 같이 응답을 반환합니다. +
추가로 ``refreshToken`` 이름을 갖는 쿠키를 삭제하고
요청한 ``AccessToken``과 ``RefreshToken``을 사용할 수 없게 합니다.
