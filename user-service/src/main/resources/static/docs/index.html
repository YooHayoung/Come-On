공통 스펙

공통 응답 스펙

Table 1. 공통 응답 스펙
이름 설명

responseTime

응답 시간을 반환합니다.

code

응답 코드를 반환합니다.

data

결과 데이터를 반환합니다.

User-Service 응답메시지는 모두 위와 같은 스펙으로 구성되어 있습니다.

code field : 응답 코드 반환값
Table 2. 응답 코드
이름 설명

BAD_PARAMETER

요청 파라미터가 잘못되었습니다.

NOT_FOUND

리소스를 찾지 못했습니다.

SERVER_ERROR

서버 에러입니다.

SUCCESS

요청이 성공하였습니다.

UNAUTHORIZED

인증에 실패하였습니다.

공통 응답 스펙의 code 필드에는 위와 같은 값들이 반환될 수 있습니다.
응답 코드는 SUCCESS인 경우를 제외하고 모두 예외가 발생한 상황입니다.

공통 응답 스펙 - 예외인 경우

Table 3. 예외 응답 스펙
이름 설명

code

API 내부에서 지정한 예외 코드를 반환합니다.

message

예외 메시지를 반환합니다.

예외가 발생한 경우에는 data 필드에 위와 같은 형식으로 발생한 예외에 대한 응답이 들어갑니다.
아래의 예시를 참고해주세요.

{
  "responseTime" : "2022-08-26T03:50:41.805042",
  "code" : "SERVER_ERROR",
  "data" : {
    "code" : 900,
    "message" : "죄송합니다. 서버 내부 오류입니다."
  }
}
errorCode field : 예외 응답 코드 반환값
Table 4. 예외 응답 코드
이름 설명

900

죄송합니다. 서버 내부 오류입니다.

901

파일이 비어있습니다. 확인해주세요.

902

파일 업로드에 실패하였습니다.

903

요청 데이터 검증에 실패하였습니다.

904

해당 식별자를 가진 리소스가 없습니다.

905

요청을 수행할 권한이 없습니다.

906

탈퇴 처리된 회원입니다.

907

인증된 사용자만이 이용 가능합니다.

예외 응답 스펙의 code 필드에는 위와 같은 값들이 반환될 수 있습니다.

1. 회원 정보 저장

POST /users

1) 정상 흐름

요청 예시

POST /users HTTP/1.1
Content-Type: application/json;charset=UTF-8
Content-Length: 142
Host: localhost:8080

{
  "oauthId" : "12345",
  "provider" : "KAKAO",
  "name" : "testName1",
  "email" : "email1@email.com",
  "profileImgUrl" : "profileImgUrl"
}

요청 필드

Table 5. 요청 필드
필드명 타입 필수값 설명

oauthId

String

true

OAuth 로그인 성공시, Provider에서 제공하는 유저 ID값

provider

String

true

OAuth 유저 정보 제공자

email

String

true

유저 이메일 정보

name

String

true

유저 이름 또는 닉네임 정보

profileImgUrl

String

유저 프로필 이미지 URL

시스템에 등록된 Provider를 지정해야 합니다. 지원하는 Provider 목록은 아래와 같습니다.

Table 6. Provider 목록
이름 설명

KAKAO

카카오

응답 예시

요청을 성공적으로 처리하면 다음과 같은 응답을 반환합니다.

HTTP/1.1 200 OK
Content-Type: application/json;charset=UTF-8
Content-Length: 389

{
  "responseTime" : "2022-08-26T03:50:41.535689",
  "code" : "SUCCESS",
  "data" : {
    "userId" : 8,

    },
    "role" : "ROLE_USER",
    "email" : "email1@email.com",
    "name" : "testName1"
  }
}

응답 필드

Path Type Description

Number

nickname

String

Number

프로필 이미지의 식별값

profileImg.imageUrl

String

프로필 이미지의 URL

2) 예외

지원하지 않은 Provider를 지정하거나, 필수값을 입력하지 않으면, 아래와 같은 오류 응답을 반환합니다.

요청 예시

POST /users HTTP/1.1
Content-Type: application/json;charset=UTF-8
Content-Length: 113
Host: localhost:8080

{
  "oauthId" : null,
  "provider" : "DAUM",
  "name" : "testName1",
  "email" : null,
  "profileImgUrl" : null
}

응답 예시

HTTP/1.1 400 Bad Request
Content-Type: application/json;charset=UTF-8
Content-Length: 288

{
  "responseTime" : "2022-08-26T03:50:41.645307",
  "code" : "BAD_PARAMETER",
  "data" : {
    "code" : 903,
    "message" : {
      "oauthId" : [ "must not be blank" ],
      "provider" : [ "지원하지 않는 Provider 입니다." ],
      "email" : [ "must not be blank" ]
    }
  }
}

응답 필드

Path Type Description

Number

Object

API 오류 메시지

message.provider

Array

필드 오류 메시지

message.oauthId

Array

필드 오류 메시지

message.email

Array

필드 오류 메시지

2. 회원 정보 조회

GET /users/{userId}

GET /users/7 HTTP/1.1
Content-Type: application/json;charset=UTF-8
Host: localhost:8080
